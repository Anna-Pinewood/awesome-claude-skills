# AI-Native Course (AIM LMS)

Доступ к материалам курса **AI-Native Sprint v3** (`ain3`) через AIM LMS Public API v1
и помощь в учёбе: разобрать лекцию/воркшоп, найти нужную сессию, сделать домашку
с опорой на материал.

Токен уже лежит в `.env` рядом со скиллом (заскоуплен ровно на `ain3`, 29 сессий).
Все команды — через один скрипт `scripts/aim.py`, запускать **`uv run`** (не голый python).

## Как запускать

```bash
uv run <skill-path>/scripts/aim.py <команда> [аргумент]
```

`<skill-path>` — каталог этого скилла. Токен подхватывается из `.env` автоматически.

## Команды

| Команда | Что делает |
|---|---|
| `list` | все сессии курса (id, дата, тип, спикер, заголовок) |
| `list <запрос>` | отфильтровать по подстроке в заголовке/типе/спикере (напр. `list лекция`, `list Смирнов`) |
| `show <id>` | детали одной сессии + summary |
| `transcript <id>` | транскрипт сессии (markdown) |
| `chat <id>` | чат сессии (markdown) |
| `material <id>` | summary + транскрипт + чат одним куском — удобно для разбора и домашки |

Примеры:
```bash
uv run <skill-path>/scripts/aim.py list
uv run <skill-path>/scripts/aim.py list practicum
uv run <skill-path>/scripts/aim.py show ain3-w1-lecture
uv run <skill-path>/scripts/aim.py material ain3-practicum-m1
```

## Рабочий процесс

1. **Найти нужную сессию.** Если пользователь не назвал точный `id` — сначала `list`
   (или `list <ключевое слово>`), выбрать сессию по заголовку/спикеру/дате.
2. **Забрать материал.** Для разбора одной сессии удобнее всего `material <id>` —
   получишь summary + транскрипт + чат сразу. Если нужно только что-то одно —
   `transcript` / `chat`.
3. **Помочь с задачей.** Домашки в этом курсе идут из лекций/практикумов и часто
   описаны прямо в summary или транскрипте сессии. Достань материал соответствующей
   недели, найди в нём формулировку задания и помогай по нему — опираясь на то, что
   реально было в материале, а не на общие знания.

## Важные нюансы

- **Транскрипты и чаты наполняются по ходу курса.** Если `transcript`/`chat` отвечает
  «ещё не выложен» (HTTP 404) — это нормально, материал просто пока не залит. Не считай
  это ошибкой скилла; работай по `summary` или предложи вернуться позже.
- **Не используй фильтр `?lab=` руками через curl** — для этого токена он даёт 403.
  Скрипт берёт весь список без фильтра (токен и так видит только `ain3`).
- **Rate limit.** У API лимит запросов в минуту. Не гоняй десятки запросов подряд в
  цикле — скрипт уже ретраит 429 с `Retry-After`, но лишние вызовы всё равно не нужны.
- **Токен — секрет.** Лежит в `.env`, который в `.gitignore`. Не выводи его в чат,
  не коммить, не вставляй в curl-примеры в открытом виде.

## Структура курса (ain3)

29 сессий типа `workshops`: preworks (`ain3-prework0X`), практикумы
(`ain3-practicum-mX`), intro, по неделям `ain3-wN-lecture` / `-practical` / `-workshop-*`
/ `-guest-*` / `-oh` (office hours), и `ain3-demo-day`. Точный актуальный список —
всегда через `list`, а не по памяти.
