# WRITE-PROTOCOL — общий протокол записи для скиллов-редакторов

Общий для всех листьев POS, которые правят страницы волта. 

## Разметка вставок

Модель записи: пишешь сам, без согласования каждой правки — ревью происходит постфактум. Каждую свою вставку (текст, буллет, строку таблицы) оборачивай в:

`<span class="pos-new" style="background:#D97757;color:#fff;padding:0 3px;border-radius:3px">…твой текст…</span>`

## После записи: откройте страницы на ревью

После записи открой каждую затронутую страницу, по одной:
`obsidian://open?vault=cloud-base&file=<путь от корня волта, url-encoded>`

## Снять подсветку

Команда `/pos прими правки <страница>` стирает обёртки `<span class="pos-new" …>…</span>`, оставляя только текст. (Позже — клик в Obsidian-плагине, см. `svod-status.md` §6.)

## Отчёт в чат

Коротко, после записи и открытия: что куда ушло (1 строка на пункт) + список открытых страниц. Это отчёт, не запрос подтверждения.
