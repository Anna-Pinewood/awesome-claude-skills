## Стратегия поиска

| Когда | Чем |
| --- | --- |
| Известен путь файла (из map.md, projects-bucket, ссылки) | `Read` или `mcp__obsidian__view file` |
| Поиск по wiki-link / связям, обход графа | `mcp__obsidian__graph` (`neighbors`, `backlinks`, `traverse`) |
| Поиск по точной фразе или ключевому слову | сначала `Bash + grep -rliE` по vault, **потом** `mcp__obsidian__vault search` для фрагментов |
| Поиск по имени файла | `mcp__obsidian__vault list` или `Bash + ls/find` |

## Известная грабля: MCP search демотит совпадения

`mcp__obsidian__vault search` в ranked-режиме иногда **не возвращает** файлы, в которых искомая фраза реально есть, особенно для русских словоформ и многословных фраз.

**Пример:** запрос «пространство проблемы» / «problem space» / «пространство» по vault → 0 прямых попаданий, при том что фраза «Пространство проблем» есть в `3 – Knowledge Pages/ML System Desing.md` и `Бабушкин – Design Document.md`. Файлы нашлись только через `grep -rliE "пространств[оа]\s+проблем" /path/to/vault`.

**Правило:** если ищешь фразу или слово и MCP-search вернул «no results» или подозрительно мало — **сразу пробуй grep по файловой системе**, не пытайся переформулировать MCP-запрос третий раз.

```bash
VAULT="/Users/olgalipina/Yandex.Disk.localized/obsidian-vault/cloud-base"
grep -rliE "<regex>" "$VAULT" 2>/dev/null
# затем grep -n -B2 -A8 для контекста на найденных файлах
```

Regex с альтернативами морфологии (`[Пп]ространств[оа]`, `проблем[аы]`) сильно надёжнее точной фразы.

## Граф

Нужен **только** когда:
- исследуешь незнакомую тему и хочешь понять, во что она встроена;
- ищешь, кто ссылается на конкретную заметку (`backlinks`);
- хочешь увидеть кластер вокруг проекта.

**Не нужен**, когда:
- путь уже известен;
- нужна фраза в тексте;
- работаешь с одной заметкой целенаправленно.

## MCP-сервер падает

Если `mcp__obsidian__*` начинает возвращать ошибки соединения — это не повод бросать задачу. Перейди на `Bash + grep + Read` по `Vault path` выше; они работают независимо от MCP. Сам перезапуск сервера — операционная задача, не работа с vault.
