# 2gis-team

Моя «команда» для работы в 2ГИС: читает за меня переписки в Mattermost, помогает формулировать ответы в моём стиле. Расширяется со временем — встречи, код, ресёрч.

## Что внутри

```
2gis-team/
├── .claude-plugin/plugin.json
├── scripts/                    ← плагин-уровневые скрипты, переиспользуются скиллами
│   ├── _mm_api.py              ←   общий слой над Mattermost REST API
│   ├── mm_read.py              ←   N сообщений из канала / ЛС
│   └── mm_fetch.py             ←   один пост по permalink
└── skills/
    └── mm-reply/
        ├── SKILL.md            ← рабочий процесс «написать сообщение в моём стиле»
        └── references.md       ← список ссылок на мои сообщения-эталоны (по тону)
```
